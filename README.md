# Bunnings Warehouse Inventory Management System

## Company
Bunnings Warehouse

## Overview
The Bunnings Warehouse Inventory System provides the employees with an inventory system 
that is more automated and efficient, and has more features than the legacy system. It 
tracks you stock levels, location of items between departments, gives automated low stock 
alerts and lets you generate order requests.
